== Description ==

Return to the blogging philosophy of WordPress - ditch the Dashboard and get writing as soon as you log in.

== Installation ==

Same as always, folks - you can download the ZIP and install via WordPress Plugins page - or, much more simply, download direct from the WP.org repository.

= Updating =

Update from your friendly neighborhood WordPress.org repository.  I don't envision many updates, the plugin is incredibly simple.
